4

Write Your First Agent

Create a Python file in the agents folder. The brainstem picks it up automatically.

~/.brainstem/src/rapp_brainstem/agents/weather_agent.py
AGENT = { "name": "weather", "description": "Get weather for any city", "parameters": { "city": {"type": "string", "description": "City name"} } } def run(city: str) -> str: return f"Weather in {city}: 72°F, sunny"

That's it. Save the file. The brainstem discovers it. The AI can now use it. No restart needed.

This is the RAPP pattern — one file, one agent. The same pattern that produced 70 customer-facing agents at Microsoft.
5

Customize the Personality

Edit the soul file to give your AI a personality:

~/.brainstem/src/rapp_brainstem/soul.md

This is a markdown file — the AI's system prompt. Write who it should be. It persists across sessions. It grows with every conversation.

The soul file is what makes RAPP different from ChatGPT. Your AI remembers. It accumulates personality. It never resets.
